# LastSurvivor_CollageGameJamGame
Last survivor - First person shooting game Created for a Seamedu (Collage) game jam

### How to play
Press W/A/S/D to move around
Press 1 to Switch Gun
Press F to Heal
Right Click to Aim
Left Click to Shoot
Press ESC to Pause
